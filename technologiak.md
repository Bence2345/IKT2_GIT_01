# Felhasznált technológiák

## HTML
- Weboldalak struktúrájának leírására szolgáló jelölőnyelv
- Böngésző értelmezi és jeleníti meg

## CSS
- A HTML oldalak megjelenésének formázására használjuk
- Színek, elrendezés, betűtípusok

## JavaScript
- Interaktív viselkedések megvalósítására
- Böngészőben futó programozási nyelv
